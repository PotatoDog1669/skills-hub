# AGENTS.md

## Project Focus

- This project values distinctive, high-quality user experience.
- Build on an established design context and push execution quality further.

## Working Rules

- Use advanced visual, motion, and interaction techniques only when they strengthen usability and brand expression.
- Preserve accessibility, responsiveness, and performance while increasing expressiveness.
- Prefer cohesive systems over one-off visual tricks.

## Quality Bar

- Enhance only after the core interface is clear and functional.
- Keep advanced effects intentional, testable, and production-ready.
