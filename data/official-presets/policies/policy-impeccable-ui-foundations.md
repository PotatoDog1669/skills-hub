# AGENTS.md

## Project Focus

- This project is design-driven and user-facing.
- Optimize for clarity, visual quality, responsiveness, and cohesive interaction design.
- Establish design context before making stylistic decisions.

## Working Rules

- Use existing design context from instructions or `.impeccable.md` before design work.
- Prefer intentional visual direction over generic safe defaults.
- Keep accessibility, responsiveness, and performance in scope for user-facing changes.
- Refine typography, spacing, copy, and interface states until the UI feels cohesive.

## Quality Bar

- Avoid generic AI-looking layouts and decorative patterns without purpose.
- Check loading, empty, error, hover, focus, and mobile states.
- Treat polish as part of implementation, not a separate afterthought.
